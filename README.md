# Highscores
Projeto de uma minimal api para inserção e consulta de pontuações.

## Documentação
Para melhor organização, a documentação foi disponibilizada na wiki do repositório, que pode ser acessada clicando [aqui](https://github.com/julioschuambach/highscores-api/wiki).

## Funcionalidades
- Inserção de uma nova pontuação, informando: nome do jogador e pontuação.
- Listagem de todas as pontuações de um determinado jogador.
- Listagem de todas as pontuações.
